#  Sales Dashboard Analysis

##  Problem Statement

Analyze sales data to identify trends, top-performing products, and regional performance to support business decision-making.

##  Dataset

* Superstore Sales Dataset (Kaggle)

##  Approach

* Data Cleaning and preprocessing
* Date conversion and feature extraction (Month, Year)
* Exploratory Data Analysis (EDA)
* Visualization using Matplotlib and Seaborn

## Analysis Performed

* Monthly Sales Trend
* Top 10 Products by Sales
* Region-wise Sales Analysis
* Category-wise Sales Analysis
* Sales vs Profit Relationship

##  Key Insights

* Technology category generates the highest sales
* Furniture and Office Supplies contribute moderately
* Some products show high sales but low profit
* Certain regions outperform others
* Monthly trends indicate seasonal patterns

##  Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
